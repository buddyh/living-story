# Story

## STORY

You are turning a prompt workflow into a durable project memory. The important
shift this window is that the work now has a concrete shape: project files,
verification rules, history snapshots, and a simple way to rerun the process.
That makes the workflow easier to trust because future updates can compare
against evidence instead of relying on recall.

The next risk is scope creep. A CLI could be useful later, but the current
version is strongest as a clear loop that any capable coding agent can run
against local repositories.
